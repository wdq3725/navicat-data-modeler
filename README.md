简单说， 就是一个让你"看着图设计数据库"的工具。不用敲 SQL 建表，也不用脑补字段之间的关系，鼠标拖几下，表、视图、外键、约束全都能可视化摆好。它支持三种标准记法——Crow's Foot、IDEF1x、UML，哪派的人都用得顺手。
